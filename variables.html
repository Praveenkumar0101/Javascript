Variables

In JavaScript, variables are used to store data values. A variable is a named container that can hold a value of a specific type, such as a number, string, or object. The value of a variable can be changed during program execution.

1:var

var is a keyword used to declare a variable and optionally initialize it to a value

The var keyword is function-scoped, meaning it’s only accessible within the function it’s declared in. If declared outside any function, it’s globally scoped.

2:Let

let is a keyword used to declare a block-scoped variable. Unlike the var keyword, which has function scope, variables declared with let are limited to the block in which they are defined

Const

const is a keyword used to declare a block-scoped variable. Unlike the var keyword, which has function scope, variables declared with const are limited to the block in which they are defined
