JavaScript has 2 main categories Datatypes :

Primitive datatypes

1.String

A string in JavaScript is a sequence of characters that can represent text, numbers, symbols, or any other type of data. A string can be created by enclosing the characters inside single quotes ('), double quotes ("), or backticks (`)

2.Number :

All JavaScript numbers are stored as decimal numbers (floating point). Numbers can be written with, or without decimals

4.Boolean

A JavaScript Boolean represents one of two values: true or false.

5.Undefined

undefined is a special value that indicates a variable has not been assigned a value or has been declared but not initialized . For example, if you declare a variable but do not assign a value to it, the variable will have the value of undefined

6.Null

null is a special value that represents the intentional absence of any object value. It is often used to indicate that a variable has no value or that an object does not exist 1. For example, if you declare a variable and set its value to null, the variable will have the value of null.


REFERENCE DATA TYPES:

1.object

An object can have properties and methods. Properties are variables that hold data values, while methods are functions that perform actions

2.array

An array is a special variable in JavaScript that can hold more than one value. It is a collection of elements, which can be of different data types such as numbers, strings, objects, and even other arrays. An array is created using square brackets [] and the elements are separated by commas

3.Function

A function can be defined using the function keyword, followed by a name, followed by parentheses (). Function names can contain letters, digits, underscores, and dollar signs. The parentheses may include parameter names separated by commas.
